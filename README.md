# Tic tac toe

This project was bootstrapped with [Create React App](https://github.com/facebook/create-react-app).

This project is deployed on GitHub Pages at https://j-mori.github.io/react-tic-tac-toe/

## Developer quick-start

1. Install [Node](https://nodejs.org/en/).
1. Clone the repo.
1. `cd` into the project root.
1. Run `npm install` to install project dependencies.
1. Run `npm start` to start the development server on http://localhost:3000/.

To run the test suite:

```bash
npm test
```

To build a static version of the app:

```bash
npm run build
```

Changes will be automatically deployed once merged with master branch.
